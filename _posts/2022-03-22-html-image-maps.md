# HTML 이미지 맵

## 이미지 맵

HTML ```<map>``` 태그는 이미지 맵을 정의합니다. 이미지 맵은 클릭 가능한 영역이 있는 이미지입니다. 영역은 하나 이상의 ```<area>``` 태그로 정의됩니다.

다음 이미지에서 컴퓨터, 전화기 또는 커피잔을 클릭해 보겠습니다.
