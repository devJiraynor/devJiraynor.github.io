# HTML 텍스트 형식 지정

## HTML 포맷 요소

서식 요소는 특수 유형의 텍스트를 표시하도록 설계되었습니다.

+ ```<b>``` - 굵은 글씨
+ ```<strong>``` - 중요 텍스트
+ ```<i>``` - 이탤릭체
+ ```<em>``` - 강조 텍스트
+ ```<mark>``` - 마크된 텍스트
+ ```<small>``` - 작은 텍스트
+ ```<del>``` - 삭제된 텍스트
+ ```<ins>``` - 삽입된 텍스트
+ ```<sub>``` - 서브스크립트 텍스트
+ ```<sup>``` - 상위 텍스트
