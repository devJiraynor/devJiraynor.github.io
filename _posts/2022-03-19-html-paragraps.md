# HTML 단락   
   
## HTML 단락   
   
HTML ```<p>``` 요소는 문단을 정의합니다.

단락은 항상 새 줄에서 시작되며, 브라우저는 단락의 앞뒤에 공백(여백)을 자동으로 추가합니다.
